# BrowserClaw
No terminal, no code, pure browser based implementation of ClaudeClaw. All data is stored locally. Powered by Gemini API. Skills powered by Make.com. Download the code and host it on any HTTPS server - Godaddy / Hostinger / Vercel or access it at browserapps.org/browserclaw/index.html. Requires free gemini api key and free make.com account.
