# BrowserClaw

BrowserClaw is a personalized, browser-based AI assistant powered by the Gemini API. It features voice input, task reminders, long-term user memory, and custom webhook integrations.

Download ths source code and upload it on any https enabled server (GoDaddy, Hostinger, Vercel, Netlify) or access it directly at browserapps.org/browserclaw/index.html. All data / keys are stored locally on your browser. Installable as a PWA on your mobile devices.

## 🚀 Getting Started

To unlock the full functionality of BrowserClaw, follow these quick steps:

### 1. Configure Your API Key
BrowserClaw requires a Gemini API key to function.
- Click the **Settings** panel (gear icon in the top right).
- Enter your **Gemini API Key**.
- *(Optional)* Customize the bot's personality and your preferred UI accent color, then click **Save Settings**.

### 2. Set Up Task Reminders
Keep track of your tasks directly within the chat!
- Type `/remind me to [task] [time]` in the chat (e.g., `"/remind me to call John tomorrow at 3pm"`).
- View all your active reminders in the **Reminders** panel (bell icon). 
- From the panel, you can also sync reminders to your Google, Outlook, or Apple Calendar.

### 3. Create Custom Connections (Webhooks)
Connect BrowserClaw to external services using custom Make.com webhooks.
- **Auto-create:** Type `/skill [what it does]` directly in the chat.
- **Manual setup:** Open the **Connections** panel (link icon) to map a custom trigger word (e.g., `/email`) to a specific webhook URL.

### 4. Manage User Memory
BrowserClaw continuously extracts important facts about you from your conversations to personalize future chats.
- Open the **User Memory** panel (user info icon) to view learned facts.
- You can manually add new facts or click **Optimize Memory** to automatically clean up and consolidate duplicates.

### 5. Voice Input
Click the microphone icon in the chat input box to speak directly to BrowserClaw for a quick, hands-free experience.
